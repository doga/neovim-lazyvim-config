# neovim-lazyvim-config

[LazyVim](https://www.lazyvim.org/) configuration files for [Neovim](https://neovim.io).

## Features

- Installs some nice-looking color schemes. Choose a color scheme by uncommenting it in `lua/plugins/colorscheme.lua`.
- Adds a sample keymap.

## Install

Once you have a working Neovim/LazyVim setup, copy the files under the `dist` directory to `~/.config/nvim/`.
It's up to you if you wish these files to overwrite your existing configuration files or not.
